TelethroidClient Documentation

Overview

The TelethroidClient class is a client for interacting with the Telegram Bot API. It allows you to send messages, get updates, and start polling for new updates.

Usage

To use the TelethroidClient class, you first need to import it:

from Telethroid import TelethroidClient

Then, you can create a new instance of the TelethroidClient class by passing in your Telegram bot token, API ID, and API hash:

client = TelethroidClient(token, api_id, api_hash)

You can then use the various methods provided by the TelethroidClient class to interact with the Telegram Bot API.

Methods

The TelethroidClient class provides the following methods:

Method Description Parameters Returns
send_message(chat_id, text) Sends a message to the specified chat.
  • chat_id (int): The ID of the chat to send the message to.
  • text (str): The text of the message to send.
 A dictionary representing the response from the Telegram API.
get_updates() Retrieves new updates from the Telegram API. No parameters. A list of updates from the Telegram API.
start_polling(handler) Continuously polls the Telegram API for updates and calls a handler function for each update.
  • handler (function): A function that will be called with each new update.
 No return value.
run() Starts the bot client. No parameters. No return value.

Examples

Here's an example of how to use the TelethroidClient class to send a message to a chat:

client.send_message(chat_id=123456789, text="Hello, world!")

Here's an example of how to use the TelethroidClient class to start polling for updates:

def handle_update(update):
message = update['message']
chat_id = message['chat']['id']
text = message['text']
client.send_message(chat_id=chat_id, text=text)
client.start_polling(handle_update)

Here's an example of how to use the TelethroidClient class to start the bot client:

client.run()

Msg Class Documentation

The Msg class is used to represent a message in a chat.

Constructor

The constructor takes three arguments:

  • message_id: an int representing the ID of the message.
  • chat_id: an int representing the ID of the chat the message was sent in.
  • text: a str representing the text of the message.

Example:

msg = Msg(123, 456, "Hello, world!")

Methods

__str__()

Returns a string representation of the message in the format "Message <message_id> in chat <chat_id>: <text>".

write(out: BytesIO, message: 'Msg')

Writes the message to a BytesIO object. The message parameter should be an instance of the Msg class.

read(data: bytes) -> 'Msg'

Reads a message from a bytes object and returns a new instance of the Msg class. The data parameter should be a bytes object containing the message data.

import_(data: Any) -> 'Msg'

Imports a message from either a bytes object or a dictionary object and returns a new instance of the Msg class. The data parameter should be either a bytes object or a dictionary object with keys "message_id", "chat_id", and "text".
